# CTA 唐奇安突破信号报告

## 架构

```
a_stock_fetcher  →  从新浪等第三方拉取数据
       ↓
a_stock_db       ←  统一 SQLite 数据库
       ↓
cta_report       ←  从数据库读取，生成报告
```

## 命令（统一入口 cli.py）

```bash
cd cta-report

# 1. 生成报告（Top 20 评分）
python3 cli.py run
python3 cli.py run --date 2026-04-07

# 2. 扫描唐奇安突破（全市场）
python3 cli.py scan
python3 cli.py scan --date 2026-04-07 --top 100
python3 cli.py scan --save

# 3. 回测（海龟唐奇安突破策略）
python3 cli.py backtest --code 000001 --start 2024-01-01
python3 cli.py backtest --codes 000001,002382 --capital 50000

# 4. 运行测试
python3 cli.py test
```

## 输出

| 目录 | 说明 |
|---|---|
| `output/` | 每日信号报告（Markdown） |
| `backtest/` | 回测报告（HTML） |

## 数据库表

| 表名 | 说明 |
|---|---|
| `cta_donchian_scan` | 每日唐奇安扫描结果 |

## 评分体系

### 信号报告评分（满分 130 分）

| 因子 | 满分 | 说明 |
|---|---|---|
| 大盘共振 | 30 | 大盘涨幅 ≥0.5% 得30分 |
| 回踩 | 40 | 相对当日最高价回撤越小越好 |
| 开盘结构 | 25 | 缺口+回调+守住开盘价 |
| 均价偏离 | 15 | 现价相对均价偏离 |
| 量价配合 | 9 | 尾盘强度+下午放量 |

### 唐奇安扫描评分（满分 90 分）

| 因子 | 满分 | 说明 |
|---|---|---|
| 突破强度 | 35 | 1~3% 最佳，太弱/太强都扣分 |
| 突破天数 | 25 | 连续突破上轨天数，1~5天最好 |
| 距下轨安全垫 | 20 | 安全垫越大下跌空间越小 |
| 量比 | 10 | 突破时放量更可靠 |

## 回测策略规则

- **信号**：收盘价突破 20 日唐奇安上轨
- **买入**：次日开盘价买入
- **止损**：买入价 - ATR × 1.3
- **止盈**：买入价 + ATR × 2.0
- **平仓**：触发止损或止盈立即平仓
- **默认初始本金**：10 万
